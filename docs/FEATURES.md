# 小呆相册 功能说明

> 本文档描述当前应用面向用户的功能，以及各功能入口、使用方式和注意事项。

---

## 一、照片库管理

### 1.1 添加/移除文件夹

- **入口**：侧边栏「照片库」→「添加文件夹」
- **功能**：
  - 选择本地文件夹加入照片库，支持多文件夹。
  - 自动检测文件夹冲突：
    - 新文件夹是已有文件夹的子目录 → 提示无需重复添加。
    - 新文件夹是已有文件夹的父目录 → 可选择用父目录替换所有子目录。
- **移除文件夹**：仅删除数据库中的索引和缩略图缓存，不会删除原始照片文件。

### 1.2 扫描照片

每个文件夹提供两个扫描选项：

| 操作 | 说明 |
|------|------|
| **扫描新增** | 默认增量扫描。只处理新增、修改或已删除的文件，已存在且未改动的文件跳过。 |
| **强制重新扫描** | 清空该文件夹在数据库中的所有索引和缩略图，重新从头扫描。 |

- 扫描过程中显示进度条、当前处理文件、已完成数量。
- 扫描完成后自动刷新照片列表、时间线、统计信息。
- 扫描完成后后台自动执行**精确去重**检测（相似去重需用户手动触发）。

### 1.3 支持的图片格式

`.jpg`、`.jpeg`、`.png`、`.webp`、`.heic`、`.heif`、`.raw`、`.cr2`、`.nef`、`.arw`

## 二、照片浏览

### 2.1 时间线视图

- **入口**：侧边栏「浏览」
- **功能**：
  - 按拍摄时间倒序排列，以年月分组。
  - 支持虚拟滚动，适合大图库。
  - 顶部时间轴可快速跳转到指定年月。
  - 显示当前选中/高亮月份。

### 2.2 筛选与选择

- **筛选条件**：
  - 文件夹
  - 拍摄日期范围
  - 是否包含 GPS 位置
  - 相机型号
- **选择模式**：
  - 进入选择模式后可多选照片。
  - 支持删除选中照片（移入回收站，使用系统 `trash`）。

### 2.3 照片详情

- 点击照片打开详情弹窗：
  - 显示大图（自动加载 medium 尺寸缩略图）。
  - 显示 EXIF 信息：相机、光圈、快门、ISO、焦距、尺寸。
  - 支持编辑拍摄日期和 GPS 坐标。
  - 支持键盘左右键切换上一张/下一张。

## 三、智能去重

### 3.1 精确去重

- **依据**：文件内容哈希（xxhash64）。
- **结果**：内容完全相同的照片会被归为一组。
- **触发方式**：
  - 扫描完成后自动后台执行（仅针对新增/删除）。
  - 去重页面手动点击「精确去重」。

### 3.2 相似去重

- **依据**：感知哈希（pHash，64 位 DCT 变换）。
- **结果**：视觉上相似的照片归为一组（汉明距离 < 10）。
- **触发方式**：去重页面手动点击「相似去重」。
- **注意**：
  - 相似去重需要为所有没有 pHash 的照片计算感知哈希。
  - 对于大图库（如 10 万张）可能需要数小时，界面上会显示进度。
  - 扫描过程中关闭应用不会丢失 pHash 计算进度，但去重分组会中断，下次启动应用会自动清理残留的相似组。

### 3.3 去重结果展示

- 重复组按组内照片数量倒序排列，数量相同按最近拍摄时间倒序。
- 组内照片按拍摄时间倒序排列。
- 系统会推荐一张「最佳保留」照片（优先有 GPS、文件更大、分辨率更高），用星星标记。
- 支持删除组内非推荐照片，或批量删除选中组的所有非推荐照片。

### 3.4 去重进度

- 在全局状态维护，切换页面后进度条不会消失。
- 完成后自动刷新去重结果。

## 四、地图展示

### 4.1 地图视图

- **入口**：侧边栏「地图」
- **功能**：
  - 在地图上显示所有包含 GPS 信息的照片。
  - 照片按位置聚合为簇（cluster），缩放级别影响聚合精度。
  - 点击聚合簇展开底部抽屉，显示该位置附近的照片缩略图。
  - 点击单张照片可在详情弹窗中查看。

### 4.2 坐标转换

- 照片内 GPS 通常为 WGS84 坐标。
- 地图瓦片使用高德地图，自动进行 WGS84 → GCJ02 偏移转换。

### 4.3 地图设置

- **入口**：设置页面 → 地图瓦片
- 支持切换瓦片源：高德地图、高德暗色、天地图。
- 天地图需要用户自行申请 API Key。

## 五、设置

### 5.1 数据目录

- 默认数据目录位于 `%APPDATA%/photovault/`。
- 可自定义数据目录（包含数据库、缩略图缓存、日志）。

### 5.2 缩略图缓存

- **入口**：设置页面 → 缓存管理
- 显示当前缩略图缓存的文件总数、占用空间、small/medium 数量。
- 支持一键清除所有缩略图缓存（不会删除原始照片，下次浏览时自动重新生成）。

### 5.3 日志

- **入口**：设置页面 → 日志
- 显示最近日志内容。
- 支持打开日志文件夹、清除日志。

### 5.4 清空数据库

- **入口**：设置页面 → 危险操作
- 清空所有照片索引、去重组果、缩略图缓存，但不会删除原始照片。

## 六、性能与体验优化

| 场景 | 优化措施 |
|------|---------|
| 大图库浏览 | 虚拟滚动 + 按需加载缩略图 |
| 缩略图加载 | small 128px 网格占位，viewport 内升级为 medium 512px |
| 地图抽屉 | 优先加载可见区域缩略图，后台逐步加载其余 |
| 扫描 | 2 并发文件处理，机械硬盘友好 |
| 缩略图生成 | 扫描期间不实时生成，扫描结束后统一后台生成 |
| 去重 | pHash 2 并发计算，BigInt 汉明距离加速 |

## 七、常见问题

### Q1：删了照片、重新扫描，库里还有？

- 早期版本存在删除分页 bug，已修复。
- 现在扫描时会一次性对比文件夹下所有 DB 记录和实际文件，删除不再存在的记录。

### Q2：相似去重很慢，能关掉吗？

- 相似去重不会自动运行，只有点击「相似去重」才会触发。
- 精确去重会在扫描完成后自动后台执行，很快。

### Q3：缩略图占空间太大？

- 可在设置页面查看缩略图缓存占用并手动清除。
- 缩略图只是缓存，清除后不影响原始照片和数据库。

### Q4：地图上的照片位置偏移？

- 确保设置中选择了正确的瓦片源。
- 国内地图瓦片（高德/天地图）使用 GCJ02 坐标，应用会自动转换。
